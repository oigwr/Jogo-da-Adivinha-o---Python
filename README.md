# 🎮 Jogo de Adivinhação em Python

Um jogo simples feito em Python onde o jogador precisa adivinhar um número secreto gerado aleatoriamente.

---

## 📌 Como funciona

* O computador escolhe um número aleatório entre **1 e 100**
* Você tem um número limitado de tentativas para acertar
* A cada erro, o jogo dá uma dica:

  * "O número é maior"
  * "O número é menor"
* O jogo termina quando você acerta ou quando as tentativas acabam

---

## 🚀 Como executar

1. Certifique-se de ter o Python instalado
2. Salve o código em um arquivo, por exemplo:

   ```
   jogo.py
   ```
3. Execute no terminal:

   ```
   python jogo.py
   ```

---

## 🎯 Objetivo

Adivinhar o número secreto no menor número de tentativas possível.

---

## 🧠 Conceitos utilizados

* Estruturas de repetição (`while`)
* Condicionais (`if`, `elif`, `else`)
* Funções
* Geração de números aleatórios (`random`)
* Entrada de dados com `input()`
